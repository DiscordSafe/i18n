Preguntas frecuentes acerca de D-Safe y sus funciones o comandos.

A continuación podrás ver una lista de preguntas frecuentes donde tal vez se encuentre tu pregunta.

## 1. El bot no responde o algo no va como debería.
Si el bot no responde, comprueba que tenga permisos suficientes tanto en el canal como en los roles que tenga, de seguir así comprueba que no hayas cambiado el prefix anteriormente y por lo tanto no te responde al prefix por defecto.

:::info Información
De persistir con el error, pregunte en el [servidor de soporte](https://dicordsafe.com/discord) y reporte el error usando ``d.reportbug Descripción del error``.
Para ver el actual prefix del bot ponga ``@D-Safe prefix``.
:::

## 4. No quiero que el bot banee a un usuario cuando uso `forceban`.
Si no quieres que un usuario sea baneado al usar el forceban, puedes ignorarlo usando el comando ``d.whitelist add UsuarioID``.

## 6. El comando de `clear` no funciona.
Si el bot no borra los mensajes revisa que has seleccionado una cantidad de mensajes después del comando, si no revisa que el bot tenga permisos de borrar mensajes y vuelve a intentarlo poniendo una cantidad entre **1-100**, si después de comprobar todo esto no lo borra seguramente sea porque discord no permite que los bots borren mensajes después de **14** días.

## 7. ¿Se puede reportar por...?
A continuación te dejo una tabla que usan los verificados (encargados de los reportes con el bot) para saber qué casos deben aceptar y cuáles no.

### Motivos aceptables:
1. Raideo.
2. Bot Spamer.
3. Selfbot.
4. Spam masivo.
5. Hacks, trampas o cuentas robadas.
6. Estafas relacionadas con discord. (nitro etc)
7. Cadenas de mensajes falsos.

### Motivos que se deben eliminar y reportar a discord:
1. Dox.
2. Acoso.
3. Contenido gráfico o pornográfia ilegal.
4. Amenazas de daño a uno mismo.
5. Evasión de baneo.
6. Seguridad de los niños.
7. Padre de un usuario.
8. Suplantación de identidad.
9. Extremismo.
10. Estafas o fraudes de forma grave.
11. Amenazas graves o comportamiento amenazante.
12. Red de hacking.
13. Spam MD.

### Preguntas frecuentes sobre los reportes.

1. ¿Se puede reportar por SpamMD?
No.

2. ¿Es considerado raideo cuando el usuario tiene administrador y borra canales, etc?
No, no se considera raideo ya que tenía permisos.

3. ¿Es considerado raideo cuando se usa a un bot?
Sí, y el bot será agregado al forceban.

## 8. ¿Cómo puedo sacar captura de pantalla y poner el enlace de la prueba?
### En ordenador:
Si su ordenador no tiene una opción de sacar capturas de pantalla puede hacerlo perfectamente instalando algún programa como ShareX o LightShot, ambos se pueden encontrar googleando un poco, después debe subir la captura a alguna web como [imgur](https://imgur.com/upload) que permite compartir el enlace.

### En móvil/celular:
Normalmente si no tiene una piedra es sencillo y depende de cada dispositivo, después solo deberá entrar a [imgur](https://imgur.com/upload) subir la imagen y poner el enlace.

## 11. ¿Cómo puedo hacer un backup de mi servidor de Discord?
Discord hace poco añadió la función de plantillas del servidor que te permite generar una plantilla de los canales, configuración, permisos, etc y copiar ese enlace para copiar tu servidor en otro nuevo. Puedes acceder a esto desde ``Ajustes del servidor > Plantillas del servidor``.

## 12. ¿Por qué x usuario está en el forceban?
Podrás ver los motivos de por qué un usuario está en el forceban usando el comando ``d.baninfo UsuarioID`` de esta manera te mostrará la información como el motivo, las pruebas y el tiempo que estará ese usuario en el forceban. Puedes ignorarlo para que no se le banee usando ``d.whitelist add UsuarioID``.
